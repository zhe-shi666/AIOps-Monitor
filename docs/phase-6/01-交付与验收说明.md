# Phase-6 交付与验收说明

更新时间：2026-04-28

## 1. 本次交付范围

1. 压测资产  
- `tools/phase-6/load/http-load.mjs`：可配置并发、时长、阈值门禁。  
- 输出：请求总量、RPS、错误率、P50/P90/P95/P99。

2. 烟测资产  
- `tools/phase-6/smoke/smoke-check.mjs`：检查 `actuator/health` 与核心业务接口。

3. 故障演练资产  
- `tools/phase-6/drill/fault-drill.sh`：支持 Redis/MySQL/Ollama 重启注入并自动做恢复后烟测。

4. 灰度发布资产  
- `tools/phase-6/release/canary-check.sh`：稳定版与金丝雀版门禁检查。  
- 输出分阶段放量建议（10% → 30% → 50% → 100%）。

5. 回滚资产  
- `tools/phase-6/release/rollback-fast.sh`：快速停 canary 并验证 stable。

6. 一键执行资产  
- `tools/phase-6/run-phase6-suite.sh`：一键执行烟测+压测（可选演练）。

## 2. 对应里程碑 Exit Criteria

1. 压测达标  
- 已具备：压测脚本 + 阈值门禁（错误率、P95） + 报告输出。

2. 故障演练通过  
- 已具备：注入脚本 + 恢复等待逻辑 + 恢复后自动烟测。

3. 上线方案、回滚方案、交接文档齐全  
- 已具备：灰度门禁脚本、回滚脚本、运维交接清单与 SOP 文档。
