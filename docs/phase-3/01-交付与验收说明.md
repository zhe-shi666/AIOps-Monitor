# Phase-3 交付与验收说明

更新时间：2026-04-28

## 1. 本次交付范围

1. 异常检测引擎（Z-Score）  
- 指标：`CPU`、`MEMORY`、`DISK`、`NET_RX`、`NET_TX`、`PROCESS_COUNT`
- 输出：异常分数、基线值、观测值、严重级别、状态、检测时间
- 实时推送：`/topic/anomalies`

2. RCA 报告落库  
- 关联调查 ID / 事件 ID / 目标 ID
- 包含报告摘要、证据链 JSON、置信度、模型名、提示词哈希

3. 模型输入输出追踪  
- 记录阶段（结构化分析、复盘草稿等）
- 记录 prompt/response、耗时、状态、错误信息

## 2. 数据库变更

迁移脚本：
- `mysql/migrations/2026-04-28_phase3_aiops_engine.sql`

新增表：

执行命令：

```bash
docker exec -i aiops-mysql mysql -uroot -proot aiops_monitor < mysql/migrations/2026-04-28_phase3_aiops_engine.sql
```

## 3. 后端接口清单

1. 异常检测
- `GET /api/anomalies`
- `GET /api/anomalies/summary`

2. 调查增强
- `GET /api/investigations/{id}/rca-reports`
- `GET /api/investigations/{id}/model-traces`
- `GET /api/investigations/{id}`（新增 `latestRcaReport` 字段）

## 4. 里程碑 Exit Criteria 对应

1. 异常检测可稳定输出  
- 已实现：`AnomalyDetectionService` 在心跳写入后实时计算并持久化异常。

2. RCA 报告包含证据链与置信度  
- 已实现：`InvestigationIntelligenceService` 生成结构化分析后写入 `rca_report`，包含 `evidence_json` + `confidence`。

3. 可追踪模型输入输出  
- 已实现：`ai_model_trace` 持久化模型请求响应、阶段、耗时、状态及错误。
